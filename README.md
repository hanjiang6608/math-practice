# math-practice

# 🎓 小学三年级数学练习 - 微信H5应用

## 📱 项目简介

一个专为小学三年级学生设计的数学练习H5应用，具有智能出题、错题分析和成绩统计功能。

## ✨ 功能特点

### 1. 智能出题系统
- **自适应算法**：根据错题类型自动调整出题比例
- **难度分级**：简单、中等、困难三档可选
- **题型丰富**：加法、减法、乘法、除法全覆盖

### 2. 学习数据分析
- **实时统计**：总题数、正确数、错误数、正确率
- **历史记录**：保存最近100道练习题
- **薄弱点分析**：可视化图表展示各题型错误率
- **学习建议**：根据错题提供针对性建议

### 3. 微信适配
- 完美适配微信内置浏览器
- 支持微信分享功能
- 响应式设计，适配各种手机屏幕
- 本地存储，数据不丢失

## 🚀 部署方法

```bash

# 2. 拖拽项目文件夹到部署区域
# 3. 自动获得链接
```

#### 4. 腾讯云/阿里云对象存储（推荐国内使用）

### 方法二：自有服务器

将 `index.html` 上传到任意Web服务器目录即可访问。

```bash
# Nginx 配置示例
server {
    listen 80;
    server_name math.example.com;
    root /var/www/math-practice;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

### 方法三：微信生态部署

## 🔧 自定义配置

### 修改标题
编辑 `index.html` 中的 `<title>` 标签：
```html

```

### 修改分享内容
在 `<meta>` 标签中修改分享信息：
```html
<meta property="og:title" content="分享标题">
<meta property="og:description" content="分享描述">
<meta property="og:image" content="分享图片URL">
```

### 修改难度设置
在JavaScript中修改 `generateQuestion()` 函数中的数字范围。

## 🐛 常见问题

### Q: 数据会丢失吗？
A: 只要不清理浏览器缓存，数据会永久保存。建议定期截图保存重要数据。

### Q: 可以多个孩子使用吗？
A: 可以，但数据会混在一起。建议每个孩子使用不同设备或浏览器。

### Q: 微信打不开链接？
A: 检查链接是否以 https:// 开头，微信要求安全链接。

### Q: 可以离线使用吗？
A: 可以！只要页面加载过一次，后续离线也能正常使用。
